# Викторина - Flask Web Application

Веб-приложение викторины с системой регистрации, авторизации, виджетом погоды и таблицей лидеров.

## Возможности

- ✅ Регистрация и авторизация пользователей
- ✅ Главная страница с прогнозом погоды на 3 дня
- ✅ Викторина с случайными вопросами
- ✅ Таблица лидеров
- ✅ Адаптивный дизайн (Bootstrap 5)
- ✅ Навигационное меню
- ✅ Футер с информацией об авторе

## Установка локально

1. Клонируйте репозиторий или скачайте файлы

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите приложение:
```bash
python app.py
```

4. Откройте браузер по адресу: http://127.0.0.1:5000

## Использование

1. **Регистрация**: Создайте аккаунт на странице регистрации
2. **Вход**: Войдите используя свои учетные данные
3. **Погода**: На главной странице введите название города для получения прогноза
4. **Викторина**: Отвечайте на вопросы и зарабатывайте баллы
5. **Лидерборд**: Смотрите свое место в таблице лидеров

## Деплой на PythonAnywhere

### Шаг 1: Создание аккаунта
1. Зарегистрируйтесь на https://www.pythonanywhere.com/
2. Выберите бесплатный аккаунт (Beginner)

### Шаг 2: Загрузка файлов
1. Откройте консоль Bash на PythonAnywhere
2. Загрузите файлы проекта (через Git или Files tab)

```bash
# Если используете git:
git clone <ваш-репозиторий>
cd <имя-папки>

# Или создайте вручную через Files tab
```

### Шаг 3: Установка зависимостей
```bash
pip3 install --user -r requirements.txt
```

### Шаг 4: Настройка веб-приложения
1. Перейдите в вкладку "Web"
2. Нажмите "Add a new web app"
3. Выберите "Manual configuration"
4. Выберите Python 3.10 (или новее)

### Шаг 5: Настройка WSGI файла
В разделе "Code" найдите файл WSGI и замените его содержимое:

```python
import sys
import os

# Добавьте путь к вашему проекту
path = '/home/ВАШ_ЮЗЕРНЕЙМ/путь/к/проекту'
if path not in sys.path:
    sys.path.append(path)

# Импортируйте приложение
from app import app as application
```

### Шаг 6: Инициализация базы данных
В консоли Bash:
```bash
cd /home/ВАШ_ЮЗЕРНЕЙМ/путь/к/проекту
python3 app.py
# Нажмите Ctrl+C после инициализации БД
```

### Шаг 7: Настройка статических файлов (опционально)
В разделе "Static files" добавьте:
- URL: /static/
- Directory: /home/ВАШ_ЮЗЕРНЕЙМ/путь/к/проекту/static/

### Шаг 8: Перезагрузка
Нажмите зеленую кнопку "Reload" в верхней части страницы Web

### Шаг 9: API ключ для погоды (опционально)
Для реального прогноза погоды:
1. Зарегистрируйтесь на https://openweathermap.org/api
2. Получите бесплатный API ключ
3. На PythonAnywhere в вкладке Web -> Environment variables:
   - Variable: WEATHER_API_KEY
   - Value: ваш_api_ключ

## Технологии

- **Backend**: Flask, Flask-Login, Flask-SQLAlchemy
- **Frontend**: HTML, CSS, Bootstrap 5, JavaScript
- **База данных**: SQLite
- **API**: OpenWeatherMap (для погоды)
